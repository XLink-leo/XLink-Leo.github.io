VSCode

VSCode 是非常强大的编辑器。强大到这种程度：

可见一斑。

不过，VSCode(下简称VSC)是出了名的难配，让很多新手们只能止步于Dev-C++这种垃圾IDE。不过这一篇，我将会让傻子都会配vsc。

Part 1 下载VSC

从这里下载。你是什么系统就安哪个系统的VSC。
应该不会有人不会安装吧
如果这都不会那就去老老实实用Dev-C++去吧。

Part 2 MinGW + Path

从这里下载 MinGW。不过现在GitHub的访问不太稳定，可能需要科学上网。安装完后点击Windows徽标，在搜索框中搜索编辑系统环境变量，在打开的窗口中选环境变量(N)...，在下面的方框中选Path这一项。点击新建(N)，将你的MinGW中的bin文件夹（例如C:\mingw64\bin）的地址填进去。之后连续点击3个确定即可。

Part 3 C/C++ 环境配置

在VSC中，所有的文件及文件夹都不能以全角字符命名。
首先，按Ctrl + Shift + X打开扩展栏，安装Chinese拓展。
然后就会弹出来一条蓝框，你点就是了。重启之后你会发现你的VSC变成了中文。
再Ctrl + Shift + X搜索这几个拓展并安装：
C/C++
vscode-luogu
Code Runner
之后，按下Ctrl + Shift + P唤醒搜索框，输入C/C++: Edit Configurations(UI)并点击。
在编译器路径这一项中填上你的编译器路径。将IntelliSense模式改为windows-gcc-x64。
在下面的语言标准中改为你想要的版本。
之后在拓展栏中右键Code Runner图标，选设置。
在打开的设置栏中勾选Run In Terminal和Save File Before Run。
再往上拉，找到Executor Map这一项，点击在settings.json中编辑，在找到的代码块中找到cpp这一行，填上你想要的编译参数。
之后再保存。
